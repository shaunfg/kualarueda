# KualaRueda

Website for the KualaRueda Cuban Salsa community in Kuala Lumpur. Built as a simple static site — no frameworks, no backend.

## Hosting

Hosted via GitHub Pages. Push to `main` and it's live.

## Updating content

**Schedule** — replace `assets/schedule.png` with a new screenshot of the updated PDF.

**Curriculum** — replace `assets/curriculum.png` with a new screenshot of the updated PDF.

**Announcements** — edit the announcement cards in `index.html` under the `<!-- ── ANNOUNCEMENTS ── -->` section.

**Location / venue info** — edit the venue cards in `index.html` under the `<!-- ── LOCATION ── -->` section.

**Hero photo** — replace `assets/IMG_4123.jpeg` with a new image (keep the same filename, or update the reference in `css/style.css`).

**WhatsApp link** — search for `chat.whatsapp.com` in `index.html` and replace the URL.

**Instagram link** — search for `instagram.com/kualarueda` in `index.html` and replace.

## Structure

```
index.html        — all page content
css/style.css     — all styles
js/main.js        — navbar scroll, mobile menu, fade-in animations
assets/           — images and PDF
```
